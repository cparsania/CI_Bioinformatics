üèóÔ∏è Basic ggplot Structure

A basic ggplot consists of: ‚úÖ ggplot(data, aes(x, y)) - Define the data and aesthetics ‚úÖ A geom_*() function - Specify the plot type ‚úÖ Additional customization layers

üìâ Bar Plot üì¶

ggplot(data, aes(x = category, fill = category)) +
  geom_bar() +
  scale_fill_discrete_divergingx() +
  theme_classic()

üéØ Scatter Plot üîµ

ggplot(data, aes(x = value, y = time, color = value)) +
  geom_point(size = 4) +
  scale_color_continuous_sequential(palette = "Inferno") +
  theme_light()

üìà Line Plot üìå

  geom_line(size = 1.2) +
  scale_color_discrete_diverging(palette = "Blue-Red 3") +
  theme_minimal()

üè∑Ô∏è Customizing Titles and Labels üè∑Ô∏è

Adding titles, subtitles, captions, and axis labels enhances the clarity of your plots. üéØ

üìù Adding Titles & Labels

ggplot(data, aes(x = category, y = value, fill = category)) +
  geom_bar(stat = "identity") +
  labs(title = "Category-wise Value Distribution",
       subtitle = "A bar plot with custom colors",
       x = "Category",
       y = "Value",
       caption = "Source: Simulated Data") +
  theme_minimal()

üé® Customizing Theme & Labels

  geom_point(size = 4) +
  labs(title = "Scatter Plot of Value vs. Time",
       x = "Value",
       y = "Time",
       color = "Category") +
  theme_light()
